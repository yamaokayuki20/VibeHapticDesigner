
# プロジェクト概要
- プロジェクト名: リアルタイム触覚生成モジュールの開発
- コアコンセプト: 自然言語から「触覚パラメータ＋ビジュアル＋サウンド」を持つオブジェクトを生成し、インタラクション時の触覚をリアルタイム生成するシステム。
- MVPターゲット: Androidアプリでのデモ稼働

# 機能要件
1. オブジェクト生成画面
   - ユーザーが自然言語でプロンプトを入力（例：「ビールジョッキつくって」）。
   - AIがビジュアルを生成し、同時にオブジェクトの物理・触覚プロパティをJSON形式で出力。
   - UI上でタップすると、サウンドサンプルと衝突時の触覚サンプルをプレビュー再生
   ![EditScreen](https://github.com/user-attachments/assets/93fbfed0-8a5e-44d7-a2c5-600bc934d96b)

2. パラメータ定義（UI上で可視化・調整可能）
   - スベスベ ー ザラザラ
   - 固体 ー 液体
   - かたい ー やわらかい
   - 重い ー 軽い
   
3. インタラクション画面
   - 編集画面の「PLAY」を押すと、オブジェクトが仮想デスクに配置される（オブジェクト追加ボタンあり）。
   - デスク上のオブジェクト同士をぶつける（あてる）と、リアルタイムで触覚と音を生成。
![PlayScreen](https://github.com/user-attachments/assets/559a5b8f-c61e-43cc-98e8-b654ec13f50e)

# 動的生成ロジックの要件（別資料参照）
- 操作中オブジェクト（インタラクタ：P1）と被操作オブジェクト（インタラクティブ：P2）の触覚パラメータの平均値（Pave）を計算。
- オブジェクトの加速度と速度を監視し、「衝突」と「それ以外（摩擦など）」の状態を区別。
- パーリンノイズをベースに、上記の状態とPaveを加味して動的な触覚波形とサウンドを生成・再生する。
